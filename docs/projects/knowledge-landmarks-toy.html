Project 03

knowledge_landmarks_toy

这条线更接近一个能继续长成研究原型的项目：局部数据很少，但全局有结构化知识 landmarks。

核心问题

当数值数据只覆盖局部区域时，全局知识 landmarks 能否帮助模型在全域更稳定地泛化。

适合先看的文件

`README -> run.py -> experiment.py -> landmarks.py -> trainer.py`

你应该抓住的点

它不是在做精确逻辑约束，而是在做“全局模糊但结构化的知识 regularization”。

源码入口

README run.py landmarks.py trainer.py 代码逐行解析
