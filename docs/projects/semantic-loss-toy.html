Project 02

semantic_loss_toy

这一版最关键的是：知识不再经过 teacher，而是直接变成输出分布上的约束损失。

核心问题

怎样把“合法输出结构”变成一个可微的 probability-mass objective。

适合先看的文件

`README -> run.py -> experiment.py -> constraints.py -> trainer.py`

你应该抓住的点

`log_satisfaction_mass` 在算的不是最大类，而是模型把多少概率质量放到了合法赋值集合上。
