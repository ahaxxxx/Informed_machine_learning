Project 01

logic_net_toy

最适合第一个看，因为它把 `rule -> teacher -> student` 这条链讲得最清楚。

核心问题

规则如何不直接替代模型，而是变成一个 teacher 分布，再蒸馏回 student。

适合先看的文件

`README -> run.py -> experiment.py -> rules.py -> trainer.py`

你应该抓住的点

规则不是硬覆盖标签，而是和 student 输出一起构造 teacher，再通过 distillation 进入训练。

源码入口

README run.py rules.py trainer.py 代码逐行解析
