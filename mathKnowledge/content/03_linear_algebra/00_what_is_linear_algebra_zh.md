# 线性代数到底在研究什么

## 1. 这部分到底在解决什么问题

线性代数研究的是：

- 哪些变化可以被“线性地”描述
- 这些变化在空间里如何组织
- 一个复杂问题能不能被拆成更简单的线性结构

## 2. 核心概念的人话解释

- 向量空间：允许线性组合的对象集合
- 线性映射：保持线性结构的变化
- 矩阵：线性映射在某组基下的坐标表示

## 3. 一个最小例子

平面上的旋转、拉伸、投影，都可以看成线性映射。

矩阵不是主角本身，而是这些映射的一种表达方式。

## 4. 最容易混淆的点

- 矩阵不等于线性代数本身
- 会算行列式，不等于懂空间结构
- 解方程组只是入口，不是全貌

## 5. 这部分和后续哪里相连

线性代数后面会自然接向：

- 高等代数
- 最优化
- 数据分析与机器学习
- 微分几何和泛函分析中的线性结构
