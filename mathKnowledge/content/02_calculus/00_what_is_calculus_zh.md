# 高数到底在研究什么

## 1. 这部分到底在解决什么问题

高数最核心的问题，不是“怎么求导”或“怎么积分”，而是：

- 一个量怎样变化
- 一个过程怎样累积
- 当量无限接近时，我们怎样严肃地描述它

## 2. 核心概念的人话解释

- 极限：研究“越来越接近”的语言
- 导数：研究“瞬时变化”的语言
- 积分：研究“累积总量”的语言

## 3. 一个最小例子

如果路程是 `s(t)=t^2`，那么：

- 导数关心的是某一时刻速度是多少
- 积分关心的是一段时间里总量怎么累起来

## 4. 最容易混淆的点

- “函数值很接近”不等于“极限存在”
- “会求导”不等于“理解导数”
- “会积分公式”不等于“理解累积”

## 5. 这部分和后续哪里相连

高数后面会自然通向：

- 数学分析
- 最优化
- 微分方程
- 概率与统计里的连续模型
