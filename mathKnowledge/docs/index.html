三门课的分工

三门一起做，但不平均用力。高数负责变化语言，线代负责结构语言，概率负责不确定性语言。它们以后会共同支撑更高阶的数学和应用理论。

Calculus

高数

从极限、导数、积分到多元微积分，先把“变化”和“累积”的语言讲清楚，再平滑过渡到数学分析。

  • 研究对象：变化与累积
  • 骨干模块：极限、导数、积分、多元
  • 后续桥接：数分、最优化、微分方程

进入高数总览

Linear Algebra

线性代数

从方程组和矩阵计算出发，但核心会往向量空间、线性映射、谱结构和几何结构推进。

  • 研究对象：空间与线性变换
  • 骨干模块：空间、映射、秩、谱、正交
  • 后续桥接：高代、优化、泛函、几何

进入线代总览

Probability

概率论

把概率从公式背诵转成随机变量、分布、条件信息和极限定理的整体思维，为统计和机器学习打稳底层。

  • 研究对象：不确定性的结构化描述
  • 骨干模块：分布、期望、条件、极限
  • 后续桥接：统计、随机过程、贝叶斯
