长期数学路线

三门基础课不是终点，而是后面更严格、更抽象、更广阔的数学训练入口。这一页先把长期地图画出来，后面每条线再逐步填细。

Phase 1: 基础地基

先把高数、线代、概率做成可回查的骨干知识库。

  • 高数：极限、导数、积分、多元微积分
  • 线代：空间、映射、谱结构、正交结构
  • 概率：分布、条件信息、极限定理

Phase 2: 严格化升级

从“会用”继续走向“会定义、会证明、会区分条件边界”。

  • 数学分析
  • 高等代数
  • 概率统计的更严格版本

Phase 3: 抽象结构

进入更高层的代数、分析和结构视角。

  • 抽象代数
  • 泛函分析
  • 最优化理论与算法
  • 模糊数学

Phase 4: 几何与前沿桥接

把分析、代数、几何和现代应用理论真正接起来。

  • 微分流形
  • 黎曼几何
  • 调和分析
  • 范畴论
  • 同调代数

这条路线为什么这样排

  • 先把本科三件套真正讲通，后面抽象训练才不空。
  • 数分和高代是从“算”和“用”走向“定义”和“结构”的关键台阶。
  • 更高阶方向不急着一口吃下，而是沿桥接关系逐步推进。

路线页的作用

它不是给别人看热闹的清单，而是给这个站定边界。以后每新增一篇内容，都应该能回答：它在这张长期地图上放在哪，接向哪里。
