线代骨干框架

这一页先把线性代数的主骨架固定下来：研究对象、章节骨架、核心结构、典型题型和后续桥接。以后会不断往这个骨架上挂更细的内容。

线性代数到底在研究什么

线性代数不是单纯研究矩阵计算，它在更深层上研究的是：

  • 哪些变化可以被线性地描述
  • 这些变化怎样作用在空间上
  • 复杂问题能不能通过线性结构被拆解、比较和分类

骨干模块顺序

模块 1：线性方程组 作为入口，先看到“解空间”和“可解性”的问题。
模块 2：向量空间与子空间 把对象从“向量数组”提升到“空间结构”。
模块 3：线性映射与矩阵表示 把矩阵放回它真正的角色：映射的坐标表示。
模块 4：基、维数、秩 建立空间复杂度和自由度的统一语言。
模块 5：特征值、特征向量、对角化 研究线性变换的内在方向和可简化结构。
模块 6：内积、正交、二次型 把长度、角度和几何结构重新接回来。

核心结构簇

  • 空间结构：子空间、基、维数
  • 变换结构：核、像、秩、同构
  • 谱结构：特征值、特征向量、对角化、Jordan 视角
  • 几何结构：内积、正交、投影、二次型

最容易混淆的点

  • 矩阵不是主角，映射和空间结构才是
  • 秩是结构量，不只是一个计算结果
  • 相似、对角化、正交化是不同层次的问题
  • 会解题不等于理解结构

典型题型簇

  • 线性方程组与解空间
  • 子空间判定与基的构造
  • 映射的核、像、秩
  • 特征值、对角化、二次型化简

后续桥接

线代会自然桥接到高等代数、抽象代数、优化、数值计算、机器学习、泛函分析和微分几何里的线性结构。
