# Informed Machine Learning Research Notes

This repository documents a structured study of informed machine learning through paper reading, note-taking, and small-scale reproductions. 

The goal is to build a **long-term, traceable research workspace** where:
- papers (PDF)
- bilingual notes (ZH / EN)
- toy implementations

are systematically organized and continuously extended.

The emphasis is on **clarity, reproducibility, and theoretical understanding**, rather than UI design.
