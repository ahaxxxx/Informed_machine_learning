# math.bozhanli.com 蓝图（中文）

## 0. 核心判断

- `math.bozhanli.com` 适合单独做，不要混进当前 `research.bozhanli.com` 仓库。
- 三门可以一起做，但不能“平均用力”，要有主次和统一模板。
- 你当前最适合的人设，不是“数学大神”，而是：
  - `跨专业进入数学后，认真重建数学地基的人`
  - `能把复杂内容讲清楚，也愿意公开补课的人`

这不是降维定位，反而是更强的真实定位。

因为它同时满足三点：

- 真实：和你当前阶段一致
- 可持续：你可以一边学一边沉淀
- 有辨识度：经管背景、跨专业进数学、公开重建体系，这本身就是故事线

## 1. 你现在的优势和短板

### 优势

- 高数、线代、概率在经管体系里都拿过满分，说明本科层面的训练基础并不差。
- 你已经有较强的自学能力和整理能力。
- 你现在做 informed ML、数学复习、知识体系搭建，其实很适合做“费曼式公开学习”。

### 短板

- 和数学专业学生相比，差距通常不在“会不会算”，而在：
  - 定义层理解
  - 证明能力
  - 抽象层迁移
  - 知识之间的结构连接
- 这些差距不是一两个月补齐的，需要长期建设。

### 结论

你不是“基础不行”，而是要从“会做题”走向“更通透地理解和组织数学”。

这件事完全值得公开做，因为它很真实，而且很有长期价值。

## 2. 站点总定位

我建议 `math.bozhanli.com` 的总站定位写成下面这个方向：

> 一个跨专业进入数学的人，公开重建自己的数学地基：  
> 从高数、线代、概率开始，一边复习，一边梳理，一边把复杂内容讲清楚。

这个定位比“系统数学教程”更适合你现在。

原因是：

- 它不装成熟
- 它允许你边学边迭代
- 它天然适合直播答疑、内容更新和长期成长

## 3. 三门可以一起做，但要按“三轨并行”组织

可以三门一起做，但不建议三门每天平推。

更稳的方式是：

### 主线：高数

- 作用：最大受众入口
- 目标：夯实分析直觉，为后续数学分析打底
- 输出重点：
  - 概念解释
  - 常见题型
  - 易错点
  - 图像直觉

### 副线：线性代数

- 作用：建立结构感和抽象感
- 目标：为高等代数、最优化、几何方向打基础
- 输出重点：
  - 线性映射视角
  - 矩阵与空间的统一
  - 特征值、对角化、二次型

### 慢线：概率论

- 作用：为统计、机器学习、随机过程打基础
- 目标：建立随机变量和分布思维
- 输出重点：
  - 随机变量视角
  - 条件概率与条件期望
  - 大数定律、中心极限定理的直觉

### 推荐节奏

- 每周主攻一门
- 另外两门做轻复习和沉淀
- 三门都持续推进，但每周只设一个“重点输出主题”

例如：

- 周一到周三：高数主攻
- 周四：线代整理
- 周五：概率整理
- 周末：答疑 / 回顾 / 站点更新

## 4. 适合你的长期数学路线

你的目标不只是补高数、线代、概率，而是最终走向更高层的理论与应用。

这很好，但不能直接按“愿望清单”推进，最好做成层级路线。

### Phase 1：重建本科核心三件套

- 高数
- 线性代数
- 概率论

目标：

- 从题型熟练，过渡到概念和结构更通透
- 为后续更严格的课程做桥接

### Phase 2：进入数学专业核心语言

- 数学分析
- 高等代数

目标：

- 真正补“定义-定理-证明”的底层能力
- 从计算型学习转入结构型学习

### Phase 3：进入抽象核心层

- 抽象代数
- 泛函分析
- 最优化算法

目标：

- 建立代数结构、空间结构、优化结构的统一视野

### Phase 4：进入几何与高级分析层

- 微分流形
- 黎曼几何
- 调和分析

目标：

- 让几何、分析、结构视角真正接起来

### Phase 5：进入更高抽象层与前沿连接

- 范畴论
- 同调代数
- 模糊数学
- 更前沿的数学应用理论

目标：

- 不是为了“学名词”，而是为了找到能支撑你研究宏图的统一语言

### 一个更现实的提醒

这些方向完全可以做，但不要把它们当成“一个阶段同时推进”的待办列表。

更好的理解是：

- 这是你的长期数学地图
- 不是你接下来三个月的学习清单

## 6. 每篇内容的固定模板

为了兼顾学习、个人 IP 和可读性，每一篇最好固定成下面这个模板：

### 模板

1. 这节到底在解决什么问题
2. 核心概念的人话解释
3. 一个最小例子
4. 最容易混淆的点
5. 题型怎么识别
6. 这部分和后续哪里相连

### 举例

如果写“导数”：

1. 导数到底想刻画什么变化
2. 为什么它不是只会求公式
3. 一个几何例子
4. `导数存在`、`连续`、`可导` 最容易混在哪里
5. 常见题型有哪些
6. 它怎样连接微分、积分、最优化

这个模板特别适合你，因为它天然就是费曼学习法。

## 7. 直播答疑可以做，但要晚于第一批内容沉淀

免费数学答疑直播是值得做的。

但最稳的顺序不是“先开播，再想讲什么”，而是：

1. 先积累第一批站点内容
2. 再基于高频问题开答疑
3. 再把直播反哺回站点

### 为什么

- 没有内容底稿，直播很容易散
- 没有主题边界，答疑会变成“什么都来一点”
- 没有回收机制，直播结束后不会沉淀成资产

### 更稳的起步方式

- 固定频率：每周一次
- 固定范围：高数 / 线代 / 概率
- 固定形式：
  - 一半讲主题
  - 一半答问题

### 最适合你的直播定位

不是：

- “我来教你们所有数学”

而是：

- “我在系统重建数学基础，这里是我的公开答疑和共学空间”

这个定位更真实，也更可持续。
